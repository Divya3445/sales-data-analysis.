# Sales Data Analysis 📊

## Project Overview

This project analyzes retail sales data to identify important business insights such as regional sales performance, profitable product categories, and sales trends over time.

The analysis is performed using **SQL for querying the dataset** and **Python for data analysis and visualization** in Jupyter Notebook.

---

## Objectives

The main objectives of this project are:

* Analyze sales performance across different regions
* Identify the most profitable product categories
* Understand sales trends over time
* Generate visual insights using data visualization

---

## Tools & Technologies

* SQL
* Python
* Pandas
* Matplotlib
* Jupyter Notebook

---

## Dataset

The dataset used in this project contains retail sales transaction records including:

* Order Date
* Region
* Category
* Sales
* Profit

The dataset helps analyze product performance and business profitability.

---

## Analysis Performed

### Sales by Region

Analyzed total sales across different regions to identify the highest performing markets.

### Profit by Category

Examined product categories to determine which category generates the most profit.

### Monthly Sales Trends

Analyzed sales over time to identify seasonal patterns and sales trends.

---

## Key Insights

* The **West region generated the highest total sales**.
* The **Technology category produced the highest profit**.
* Some categories show **high sales but lower profitability**.
* Sales patterns indicate **seasonal trends in certain months**.
